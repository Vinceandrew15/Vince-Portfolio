TIGOM System Design.

Technical implementation of the Budget Planner Web Application.

User Interface & Prototyping

Figma Vue.js

Designed a comprehensive, user-friendly interface using Figma to map out user journeys and expense tracking workflows. The final frontend was developed using the Vue.js framework to ensure a reactive and seamless single-page experience.

Server-Side Infrastructure

Node.js PostgreSQL
  • 01. Built a robust back-end server using Node.js to manage financial data processing and secure user sessions.
  • 02. Utilized PostgreSQL for the primary database to ensure reliable storage of user expenses and budget records.

Hosting & Production

Hostinger Web Hosting

Successfully deployed the application using Hostinger, managing the production environment to keep the service accessible for active budget management.
