IoT Warehouse Architecture.

Technical breakdown of the Coffee Bean Warehouse Monitoring System.

Hardware & Data Acquisition

ESP8266 DHT11

Developed a specialized hardware unit utilizing the DHT11 sensor to capture precise temperature and humidity metrics. The ESP8266 microcontroller serves as the primary gateway, processing local sensor data before transmission.

Backend & Alert Systems

Node.js Semaphore API
  • 01. Engineered a Node.js server to handle API requests and facilitate real-time data transfer from the hardware units.
  • 02. Integrated the Semaphore API to automate SMS notifications, alerting personnel instantly if environmental abnormalities are detected.

Forecasting & Storage

PostgreSQL Python Flask

Implemented PostgreSQL to maintain a structured, high-integrity historical record of all warehouse metrics.

Predictive Logic
